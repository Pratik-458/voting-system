Decentralized Voting Dapp Documentation

Overview

The Decentralized Voting Dapp allows users to vote on predefined proposals using the Ethereum blockchain. This application ensures transparency, security, and immutability of the voting process, with votes stored on a public blockchain.

How to Use the Dapp

  1. Install MetaMask: Ensure you have the MetaMask browser extension installed and connected to an Ethereum network (Ganache for local development).
  2. Connect Wallet: Click the "Connect Wallet" button on the homepage. This will link your MetaMask account to the Dapp.
  3. Vote: Once connected, the available proposals will be displayed. Click on a proposal to vote.
  4. View Results: The results will update in real-time, showing the number of votes each proposal has received.

Features

  • Secure Voting: All votes are stored immutably on the Ethereum blockchain, ensuring transparency and security.
  • Real-Time Results: Votes are tallied and updated in real-time, giving users instant feedback on the voting process.
  • MetaMask Integration: The Dapp integrates with MetaMask, allowing users to securely connect and vote using their blockchain account.
  • Smart Contract Auditing: The Solidity smart contracts have been audited using Slither to ensure security and prevent vulnerabilities.

Technical Specifications

  • Blockchain: Ethereum (Tested locally with Ganache)
  • Smart Contract Language: Solidity (version 0.8.x)
  • Front-End Technologies: HTML, CSS, JavaScript (Web3.js for blockchain interaction)
  • Wallet Integration: MetaMask for wallet connection and transaction signing
  • Security Audit: Smart contracts are audited using Slither for common vulnerabilities.

Smart Contract Details

The smart contracts used in this Dapp handle the following:

  • Proposal Management: Each proposal is stored on the blockchain with a name and vote count.
  • Voting Mechanism: Users can cast a vote on a proposal, and the contract ensures that no double voting occurs.
  • Results: The contract provides real-time access to the voting results.

Get Started

To get started with the Dapp, visit the Voting Page and follow the instructions to connect your wallet and vote!
